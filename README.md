que es cliente-servidor 
El modelo cliente-servidor es una arquitectura donde un cliente envía peticiones de servicios o datos, y un servidor las recibe, 
las procesa y devuelve una respuesta. Un mismo servidor puede atender simultáneamente a muchos clientes conectados por una red, 
compartiendo sus recursos entre todos ellos. El cliente inicia siempre la comunicación de forma activa, mientras que el servidor permanece a la espera de solicitudes 
y responde de forma pasiva.

Diferencia entre proceso e hilo
Un proceso es un programa en ejecución que tiene su propio espacio de memoria y recursos aislados del resto, por lo que es más pesado de crear y gestionar. 
Un hilo es una unidad de ejecución dentro de un proceso que comparte la misma memoria y recursos con otros hilos del mismo proceso. Esto hace que los hilos sean más ligeros,
rápidos de crear y eficientes para trabajar en paralelo. En resumen, los procesos aíslan mejor, y los hilos permiten más concurrencia y rendimiento dentro de ese proceso.
